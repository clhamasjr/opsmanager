# OpsManager

Sistema de Gestão de Digitações com Supabase.

## Deploy no Vercel

1. Suba este repositório no GitHub
2. Acesse [vercel.com](https://vercel.com) e conecte seu GitHub
3. Importe este repositório
4. O Vercel detecta automaticamente que é um projeto Vite
5. Clique em **Deploy**
6. Pronto — seu sistema estará online

## Desenvolvimento local

```bash
npm install
npm run dev
```

Acesse http://localhost:5173
